Getting Started

What you can do with our APIs:

  1. Control the Car: Remote controls for honk horn, blink lights, engine on/off, remote turn on/off the engine, etc.
  2. Know the Driver: Search by driver/subscriber data, add new drivers, use payment info.
  3. Know the Car: Get both dynamic and static data about the car, add new cars.
  4. Emulate Car: Change diagnostic data, get car status.


Ready to start building?

  • Base URL http://luigi.hack.att.io
  • API REST server running on port 3000
  • TCU Car Simulator running on port 80
